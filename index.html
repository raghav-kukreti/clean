Cle.an

A clean, simple framework

Typography Forms Download

Cle.an is an extremely simple front-end framework which contains useful defaults, classes and typography. It was built using the powers of SASS and HTML5.

It has a flat color palette and sets the font-sizes, typography defaults, paddings and margins to provide a hassle free coding experience.

Developed by Raghav Kukreti
